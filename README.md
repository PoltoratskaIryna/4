# Collatz Parallel Calculation in Docker

## Запуск у Docker

1. Побудуйте образ:

    docker build -t collatz .

2. Запустіть контейнер:

    docker run --rm collatz

## Опис
- Програма генерує числа від 1 до 10 000 000 і паралельно обчислює кількість кроків до 1 за гіпотезою Колатца для кожного числа.
- Кількість потоків можна змінити у homework4.py (змінна `num_threads`).
- Після завершення обчислень виводиться середня кількість кроків та загальний час виконання.
